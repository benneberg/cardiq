ABOUT BRUTALLEARN

WHAT IS THIS?

BrutalLearn is an AI-powered learning platform that creates personalized, bite-sized lessons on any topic you choose.

HOW IT WORKS

  1. Set up your Groq API key in settings
  2. Create a new topic (e.g., "GitHub Copilot in VS Code")
  3. Define your learning goals and audience
  4. Let AI generate expert lessons for you
  5. Learn at your own pace, mark cards as done
  6. Chat with an AI expert when you need help

FEATURES

  • Microlearning: Bite-sized, manageable lessons
  • Progress Tracking: Visual progress bar and completion status
  • AI Expert Chat: Ask questions anytime
  • Daily Tips: Random lesson each day for reinforcement
  • Personal Notes: Add your own annotations
  • Multiple Topics: Switch between learning paths
  • Search & Filter: Find what you need quickly

PERFECT FOR

Corporate training, onboarding, skill development, exam prep, or personal learning journeys.
