About this app

This is a neo-brutalist, mobile-first single page app that uses Groq's OpenAI-compatible inference endpoints to generate role-based lesson cards and provide a chat professor. Add your API key in Settings, fetch models, create a Topic (tell the AI what to include and who the students are), then generate structured lesson cards. Cards can be marked done to gamify progress.

  1. Open Settings → paste Groq API key → Fetch models
  2. Create a Topic, include "what to include" and "who are students"
  3. Generate lessons → Wait for AI to return structured JSON cards
Settings
